# herai

💻 Install
-------

Create a virtual environment and activate it:

```
python3 -m venv venv --clear
. venv/bin/activate
```

Install dependencies:

```
python3 -m pip install flask==2.2.2 livermask==1.4.0
```
    
🚀 Run
---

Start the server:

```
python3 main.py
```

Open http://127.0.0.1:5000 in a browser.
